## 3. Как мы используем ваши данные

### 3.1. Основные цели:

- ✅ **Публикация афиш** — размещение одобренных мероприятий в канале
- ✅ **Модерация** — проверка контента перед публикацией
- ✅ **Статистика** — предоставление пользователям их статистики
- ✅ **Рассылки** — еженедельные подборки по пятницам (при согласии)
- ✅ **Улучшение Бота** — анализ использования для развития функционала

### 3.2. Автоматические решения:

- Бот **не принимает автоматических решений**, влияющих на пользователей
- Все публикации проходят **ручную модерацию**
- Пользователь может **отозвать согласие** на рассылку в любой момент

## 4. Хранение и защита данных

### 4.1. Где хранятся данные:

| Тип данных | Место хранения | Срок хранения |
|-----------|---------------|---------------|
| **База данных Бота** | VPS-сервер (Беларусь) | Бессрочно, до удаления |
| **Фотографии афиш** | Серверы Telegram | Бессрочно, до удаления (согласно с политикой Telegram) |
| **Логи действий** | VPS-сервер | 30 дней |
| **Резервные копии** | VPS-сервер | 7 дней |

### 4.2. Меры защиты:

- 🔒 **Доступ к базе данных** — только у Разработчика
- 🔒 **HTTPS-соединение** — все данные передаются через защищённый канал
- 🔒 **Регулярные бэкапы** — резервное копирование базы данных
- 🔒 **Логирование доступа** — фиксация всех действий с данными
- 🔒 **Минимизация данных** — собираем только необходимое для работы
- 🔒 **Защита VPS** — фаервол, SSH-ключи, регулярные обновления

### 4.3. Доступ третьих лиц:

| Третья сторона | Доступ к данным | Причина |
|---------------|----------------|---------|
| **Telegram** | Полные данные | Платформа для работы Бота |
| **Разработчики** | Полные данные | Разработка, обновление, поддержка Бота |
| **Модераторы** | Данные афиш и авторов | Проверка контента |
| **Хостинг-провайдер** | ✅ Технические данные | Размещение VPS-сервера |
| **Другие лица** | ❌ Нет доступа | — |

## 6. Публикация контента

### 6.1. Пользовательский контент:

- Отправляя афишу, вы **гарантируете**, что:
  - ✅ Имеете права на контент **ИЛИ**
  - ✅ Афиша находится в **публичном доступе** (открытые источники, соцсети, сайты организаторов)
- Вы **разрешаете** публикацию афиши в канале Бота
- Вы можете запросить **удаление публикации** в любой момент
- Вы **несёте ответственность** за соблюдение авторских прав

### 6.2. Анонимность:

- При публикации можно выбрать **анонимный режим**
- В анонимном режиме имя пользователя **не указывается**
- Модераторы всё равно видят автора для проверки

### 6.3. Модерация:

- Все афиши проходят **ручную модерацию**
- Афиша может быть **отклонена** без публикации
- Причина отклонения сообщается пользователю
